# 🚀 快速测试指南

## ✅ API Key 已配置

您的 DeepSeek API Key 已成功配置到 `index.html` 中。

## 🧪 测试步骤

### 方法一：完成测试后自动生成（推荐）

1. **打开网站**：在浏览器中打开 `index.html`

2. **完成体质测试**：
   - 点击"开始答题"
   - 完成所有36道题目
   - 查看测试结果

3. **填写个人信息**：
   - 滚动到"AI 中医顾问"部分
   - 填写您的年龄（例如：26）
   - 选择性别（男/女）
   - **系统会自动生成个性化方案**（约3-5秒）

4. **查看方案**：
   - 方案会自动显示在AI咨询对话框中
   - 包含完整的养生建议

### 方法二：手动触发生成

1. **选择体质类型**：
   - 在"AI 中医顾问"部分
   - 点击选择您的体质类型（例如：气郁质）

2. **填写个人信息**：
   - 填写年龄和性别

3. **点击生成按钮**：
   - 点击"✨ 生成个性化方案"按钮
   - 等待3-5秒

4. **查看方案**：
   - 个性化方案会显示在对话框中

## 🔍 检查功能是否正常

### 1. 检查控制台
打开浏览器开发者工具（F12），查看控制台：

**正常情况应该看到：**
```
✅ TCMDeepSeekService 初始化成功，API Key 已配置
🤖 正在生成个性化养生方案...
✅ 个性化方案生成完成，耗时：XXXXms
```

**如果看到错误：**
- `⚠️ DEEPSEEK_API_KEY 未配置` → 检查 API Key 配置
- `API请求失败: 401` → API Key 无效
- `API请求失败: 429` → 请求频率超限，稍后重试

### 2. 检查方案内容
生成的方案应包含：
- ✅ 体质分析
- ✅ 饮食建议（3条）
- ✅ 生活方式建议（3条）
- ✅ 运动建议（3条）
- ✅ 穴位推荐（至少2个）
- ✅ 四季调养要点
- ✅ 年龄专项建议（如果适用）
- ✅ 注意事项

### 3. 检查特殊功能
- **围绝经期女性（45-52岁）**：应包含疏肝滋肾专项建议
- **中年男性（40-56岁）**：应包含补肾固本专项建议

## 🐛 常见问题

### Q1: 方案没有自动生成？
**A:** 检查是否：
1. 已填写年龄和性别
2. 已选择或完成体质测试
3. API Key 已正确配置
4. 网络连接正常

### Q2: 显示"API Key 未配置"？
**A:** 
1. 检查 `index.html` 中的 `window.DEEPSEEK_API_KEY` 是否已设置
2. 刷新页面重试
3. 检查浏览器控制台是否有错误

### Q3: 显示"请求超时"？
**A:**
1. 检查网络连接
2. DeepSeek API 可能暂时不可用，稍后重试
3. 检查 API Key 是否有效

### Q4: JSON 解析失败？
**A:**
1. 查看控制台错误信息
2. 可能是 API 返回格式异常
3. 尝试重新生成
