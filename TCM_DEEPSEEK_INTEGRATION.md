# TCMDeepSeek 个性化方案集成说明

## 📋 功能概述

已成功将 TCMDeepSeekService 集成到网站的 AI 咨询模块中。测试结果后，如果用户填写了年龄和性别，系统会自动调用 DeepSeek API 生成个性化的中医养生方案。

## ✨ 主要功能

1. **自动生成个性化方案**：测试完成后，如果用户已填写年龄和性别，自动生成个性化方案
2. **手动触发生成**：用户可以通过"生成个性化方案"按钮手动触发
3. **完整的方案展示**：包括体质分析、饮食建议、生活方式、运动建议、穴位推荐、四季调养等

## 🔧 配置步骤

### 1. 配置 API Key

在 `index.html` 中找到以下代码段：

```html
<!-- DeepSeek API 配置 -->
<script>
  // 配置 DeepSeek API Key（请替换为您的实际 API Key）
  window.DEEPSEEK_API_KEY = ''; // 请在此处填入您的 DeepSeek API Key
</script>
```

将您的 DeepSeek API Key 填入 `window.DEEPSEEK_API_KEY`。

**⚠️ 安全提示**：
- 在生产环境中，建议通过后端代理 API 调用，避免在前端暴露 API Key
- 如果必须在前端使用，请确保 API Key 有使用限制和监控

### 2. 获取 DeepSeek API Key

1. 访问 [DeepSeek 官网](https://www.deepseek.com/)
2. 注册账号并登录
3. 在控制台中创建 API Key
4. 复制 API Key 并配置到网站中

## 🎯 使用流程

### 自动生成（推荐）

1. 用户完成体质测试
2. 系统显示测试结果
3. 用户滚动到 AI 咨询部分
4. 用户填写年龄和性别
5. **系统自动检测并生成个性化方案**（如果已填写年龄和性别）

### 手动生成

1. 用户选择体质类型（或完成测试）
2. 用户填写年龄和性别
3. 点击"✨ 生成个性化方案"按钮
4. 系统生成并显示个性化方案

## 📊 方案内容

生成的个性化方案包含以下内容：

1. **体质分析**：结合年龄、性别的综合评估
2. **饮食建议**：具体食材和功效说明
3. **生活方式建议**：具体可操作的建议
4. **运动建议**：适合该体质的运动类型和强度
5. **穴位推荐**：穴位名称、位置、功效、按摩手法
6. **四季调养要点**：春夏秋冬的调养重点
7. **年龄专项建议**：基于生命周期的特殊建议
8. **注意事项**：重要的健康提醒

## ⚙️ 技术细节

### API 调用参数

- **Model**：`deepseek-chat`
- **Temperature**：0.6（平衡创造性与医学准确性）
- **Response Format**：`json_object`（强制 JSON 输出）
- **Max Tokens**：2000
- **Timeout**：30秒

### 错误处理

- API Key 未配置：显示提示信息，不影响其他功能
- 用户未填写信息：提示用户填写年龄和性别
- API 调用失败：显示错误信息，用户可继续使用其他功能
- JSON 解析失败：显示错误信息，提供降级方案

## 🐛 故障排查

### 问题：方案未自动生成

**可能原因**：
1. API Key 未配置
2. 用户未填写年龄或性别
3. API 调用失败

**解决方法**：
1. 检查 `window.DEEPSEEK_API_KEY` 是否已配置
2. 提示用户填写年龄和性别
3. 查看浏览器控制台的错误信息

### 问题：API 调用超时

**可能原因**：
1. 网络连接问题
2. API 服务暂时不可用
3. 请求参数过大

**解决方法**：
1. 检查网络连接
2. 稍后重试
3. 检查 API Key 是否有效

### 问题：JSON 解析失败

**可能原因**：
1. API 返回格式不正确
2. 响应包含非 JSON 内容

**解决方法**：
1. 检查 API 响应内容
2. 查看控制台错误信息
3. 联系技术支持
